# Templates

Starter agents for developers, hackathons, and workshops. A template folder exists ONLY when it runs; no placeholders, ever.

| Template | What it is | Start |
|---|---|---|
| [agentathon-starter](agentathon-starter/) | A research notebook agent with real H-MEM memory; CLI you replace with your own surface | `npx degit meterless/meterless/templates/agentathon-starter my-agent` |

Contributing a template: it must run on a clean machine with documented steps, use at least one engine for real (no faked engine behavior), and keep the two-minute quickstart honest.
