## Positioning

H-MEM is a hierarchical human-memory architecture for agents: short-term, working, and long-term memory with mining, retrieval, dreaming, sleep, trust, and conflict resolution.

## Message

Do not sell "AI memory." Sell living memory:

- tiered storage with a forgetting curve,
- explicit, traceable reinjection,
- human-reviewed synthesis (dreaming),
- preview-first maintenance (sleep),
- append-only audit (the trust ledger),
- audited contradiction handling.
